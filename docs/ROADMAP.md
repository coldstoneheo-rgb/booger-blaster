# Development Roadmap

## Phase 1: Prototype (Current)
**Goal**: Validate core "Fun" factor of Rub & Flick mechanics.
- [ ] **Core Input**: Implement `InputSystem` to distinguish Rubbing vs Flicking.
- [ ] **Physics**: Implement `ProjectileSystem` with variable size/speed.
- [ ] **Basic Gameplay**: Endless spawn of "Basic Fly".
- [ ] **UI**: Simple score display, debug info for gestures.

## Phase 2: Core Loop & Juice
**Goal**: Make it feel good.
- [ ] **Visuals**: Add Seungjae's face states (Idle, Pick, Shock, Battle).
- [ ] **Feedback**: Particle effects for "Splat", "Charge", "Shoot".
- [ ] **Audio**: Sound FX for rubbing, flicking, splatting.
- [ ] **Intro**: Animated intro sequence.

## Phase 3: Progression & Meta
**Goal**: Retain players.
- [ ] **Levels**: Different bug waves.
- [ ] **Upgrades**: "Sticky Fingers", "Power Sneeze".
- [ ] **Customization**: Unlocking different finger skins / nail polish?

## Phase 4: Release
- [ ] Mobile optimization (Viewport meta, PWA support).
- [ ] Performance tuning.
